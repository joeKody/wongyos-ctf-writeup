## Hello Crypto [28 pts]

สวัสดีคุณ Crypto คุณรู้จักฉันมั้ย

<details>
    <summary>Solution</summary>

- TLDR: `From base64 -> From base62`
- ในไฟล์จะพบกับบางอย่างที่ผมสงสัยว่าเป็น base64 จึงนำไป decode from base64 ใน cyberchef
- เมื่อ decode แล้วก็ยังเจอกับอะไรไม่รู้ ที่น่าจะเป็น base บางอย่าง
- ผมลองใช้ทุก base และพบว่า base62 เป็น base ที่นำไปใช้แล้วพบกับ flag!

</details>
